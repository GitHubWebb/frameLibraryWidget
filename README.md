## 引用方式
## 注:从2.0-开始支持AndroidX
>1.需要在根Build.gradle中配置jitpack依赖
>>allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
	
>2.在需要调用的module	dependencies 中,加入
>>implementation com.github.GitHubWebb.frameLibraryWidget:frameLibraryWidget:2.0.6-androidx
